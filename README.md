# A Moving & Packaging Services

  Professional movers and packers website for Dubai/UAE market.

  ## Features
  - Full single-page React app (Vite + TypeScript + Tailwind CSS)
  - Instant price calculator (Studio AED 800, AED 1,200/bedroom)
  - WhatsApp booking integration (+971508199686)
  - Image gallery with lightbox
  - Services, Fleet, Packing Materials, Testimonials, FAQ sections
  - Google Maps embed (Sheikh Zayed Road, Dubai)
  - Light theme with navy (#0B1F3A) and amber (#F59E0B) brand colors

  ## Tech Stack
  - React + Vite + TypeScript
  - Tailwind CSS v4
  - Framer Motion
  - Lucide React icons
  - shadcn/ui components
  - pnpm monorepo workspace

  ## Setup
  ```bash
  pnpm install
  pnpm --filter @workspace/moving-services run dev
  ```
